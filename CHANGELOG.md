# Changelog

## 1.0.0
- Promoted extension and docs to stable `v1.0.0` release.
- Added syntax grammar contributions for `.eta` and `.ejs` to improve editor highlighting.
- Bundled runtime formatter dependencies into VSIX for reliable activation in VS Code.
- Updated README, release guide, and website with full setup, troubleshooting, and usage documentation.
- Added explicit project identity details: Extension developer `KEYHAN`, GitHub repository, and website links.

## 0.1.0
- Initial monorepo scaffold for ETA/EJS Prettier plugin and VS Code formatter extension.
- Added baseline parser/printer for `.eta` and `.ejs`.
- Added document/range formatting provider, tests, and CI workflows.
