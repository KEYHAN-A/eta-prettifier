# Security Policy

## Supported Versions

This project is pre-1.0. Security fixes are applied to the latest `main` branch.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Send details to: `security@keyhan.info`

Include:

- summary and impact
- reproduction steps
- affected files/versions
- suggested remediation (optional)

## Response Expectations

- Initial acknowledgment target: within 72 hours
- Follow-up with triage status after reproduction
- Coordinated disclosure after fix is available
